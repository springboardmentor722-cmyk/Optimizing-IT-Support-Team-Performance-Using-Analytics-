## Milestone 2 - Exploratory Visualization and Cluster Insights

## Overview

This milestone focuses on performing exploratory data analysis to discover patterns in IT support ticket data and analyzing issue clusters using channel-based grouping and similarity scores. A total of 8 visualizations were generated to support insight discovery.

---

## Module 3 - Exploratory Visualization

The cleaned dataset was analyzed to understand ticket distribution patterns across multiple dimensions.

**Visualizations Created**

- Ticket Type Distribution
- Top Categories and Clusters by Frequency
- Ticket Priority vs Status

**Key Findings**

- Ticket types are evenly distributed across all 5 categories, each holding approximately 20% share
- Medium priority tickets lead at 25.88%, followed by Critical at 25.14%
- Pending Customer Response accounts for 34.02% of all ticket statuses
- Email channel handles the highest ticket volume with 2,143 tickets

---

## Module 4 - Similarity and Cluster Insights

Ticket channels such as Email, Phone, Chat, and Social Media were used as cluster proxies. Performance and similarity scores were analyzed across each cluster.

**Visualizations Created**

- Cluster Cohesion Analysis
- Similarity Distribution Across Clusters
- Ticket Volume by Cluster
- Cluster vs Ticket Type Distribution
- Cluster vs Ticket Priority Distribution
- Efficiency Analysis: Similarity vs Resolution Time
- Performance Gap Analysis
- Detailed Similarity Spread per Cluster

8 visualizations were successfully generated covering ticket distribution, cluster analysis, similarity scores, and performance gaps. The dataset is now ready for performance trend analysis in Milestone 3.
