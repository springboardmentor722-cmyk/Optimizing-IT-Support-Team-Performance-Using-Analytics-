## Milestone 4 - Dashboard Development and Documentation

## Overview

This milestone focuses on building interactive dashboards using Power BI and Streamlit, combining all project insights into visual performance reports.

---

## Power BI Dashboard

An interactive Power BI dashboard was developed to visualize key performance indicators and support data-driven decision making.


**Key KPIs**

- Total Tickets
- Pending Tickets
- Average Resolution Time
- Response Efficiency
- Best Performing Cluster

**Visualizations**

- Resolution time by cluster
- Ticket distribution by priority
- Issue category trend analysis
- Cluster similarity comparison
- Geographic ticket distribution

---

## Streamlit Dashboard

An interactive live dashboard was built using Streamlit providing real-time filtering and performance monitoring.

Dashboard File: (Supportlytics_Dashboard.py)

**To run the dashboard locally:**
```
streamlit run Supportlytics_Dashboard.py
```

**Dashboard Includes**

- Ticket status distribution
- Issue category trend
- Resolution time analysis
- Geographic insights
- Unresolved high priority ticket tracking

---

## Tools Used

- Python, Pandas, Plotly, Streamlit
- Power BI Desktop

---

## Outcome

Both Power BI and Streamlit dashboards were successfully developed combining all performance KPIs and geographic insights.
