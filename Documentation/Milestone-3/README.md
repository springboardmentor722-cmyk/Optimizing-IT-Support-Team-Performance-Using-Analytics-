## Milestone 3 - Performance Trend Analysis and Geographic Insights

## Overview

This milestone focuses on analyzing resolution performance across priorities, ticket types, regions, and support channels. Module 5 was implemented using Python and Jupyter Notebook, Modules 6 was completed using Power BI, and Module 7 is an interactive live Streamlit dashboard.


---

## Module 5 - Performance Trend Analysis

The cleaned dataset was used to analyze resolution time patterns across multiple dimensions. A total of 8 visualizations were generated.

**Visualizations Created**

- Avg Resolution Time by Priority
- Avg Resolution Time by Ticket Type
- Team Efficiency: Avg Resolution by Cluster
- Avg Resolution Time by Region
- Resolution Time Distribution by Priority
- Resolution Time Distribution by Ticket Type
- Monthly Avg Resolution Time Trend
- High Priority Unresolved Tickets

**Key Findings**

- Average resolution time across all closed tickets is 7.75 hours
- Total of 2,769 tickets were closed out of 8,469 tickets
- High priority unresolved tickets were identified for immediate attention
- Monthly resolution trends were tracked to identify performance improvements over time

---

## Module 6 - Geographic and Category Level Insights

Geographic insights were developed using Power BI. Region, Latitude, and Longitude columns were programmatically generated to enable geographic analysis across 7 global regions including North America, Latin America, Europe, Africa, Middle East, Asia Pacific, and South Asia.

**Visualizations Created in Power BI**

- Heatmap of ticket concentration by region
- Geographic map of issue categories using latitude and longitude
- Cluster size vs performance score comparison
- Category level performance analysis across regions


---

## Module 7 - Interactive Streamlit Dashboard

An interactive live dashboard was built using Streamlit combining all performance KPIs and geographic insights into a single interactive report.


**Dashboard KPIs Included**

- Average Resolution Time
- Most Frequent Issue Categories
- Cluster Performance by Channel
- Regional Ticket Distribution
- High Priority Unresolved Tickets

---

## Tools Used

- Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
- Power BI Desktop

## Outcome

Performance trends were successfully analyzed across all priorities, ticket types, and regions. Geographic visualizations, an interactive Power BI report, and a live Streamlit dashboard were created to provide actionable insights for IT support optimization.
