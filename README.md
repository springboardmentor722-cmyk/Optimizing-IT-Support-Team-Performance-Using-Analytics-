# Optimizing IT Support Team Performance Using Analytics (Supportlytics)

## Project Overview
This project focuses on analyzing IT support ticket data to identify performance bottlenecks, optimize resolution time, and improve overall service efficiency using data-driven insights.

## Objectives
- Analyze IT support ticket data
- Improve resolution efficiency
- Identify key issue categories
- Build an interactive dashboard

## Key Features
- Data Cleaning and Feature Engineering
- Exploratory Data Analysis (EDA)
- Cluster and Similarity Analysis
- Performance Analysis
- Power BI Dashboard

## Tech Stack
- Python (pandas, numpy)
- matplotlib, seaborn, plotly
- Power BI

## Key Metrics
- Average Resolution Time
- Ticket Volume
- Priority Distribution
- Cluster Similarity Score

## Insights
- High-priority tickets take longer in some regions
- Repetitive issues form large clusters
- Performance gaps across teams

## Conclusion
Data-driven analytics helps optimize IT support performance and improve efficiency.
## Future Improvements
- Real-time ticket streaming analysis
- Predictive modeling for ticket resolution time
- AI-based ticket classification
- Automated recommendation systems
