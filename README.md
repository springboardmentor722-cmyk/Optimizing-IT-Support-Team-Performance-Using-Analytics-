# IT Support Team Performance Analytics

# 📌 Project Overview
This project focuses on analyzing IT support ticket data to evaluate support team performance, identify operational bottlenecks, and improve resolution efficiency. Using data analytics and visualization techniques, the project provides actionable insights to enhance service quality and optimize workflows.

---

# 🎯 Objectives
- Analyze IT support ticket data  
- Identify performance trends and inefficiencies  
- Optimize ticket resolution time  
- Understand patterns in ticket types, priorities, and categories  
- Improve overall support service efficiency  

---

# 📊 Key Insights
- Average resolution time is approximately **7.5 hours**  
- High-priority tickets require faster handling but often take longer to resolve  
- Certain channels (Chat/Email) perform better in resolving tickets quickly  
- Billing and technical issues are the most frequent categories  
- Resolution efficiency varies across ticket types and channels  
- High-priority unresolved tickets highlight potential performance gaps  

---

# 📊 Dashboard Features
- KPI Cards (Total Tickets, Avg Resolution Time, High Priority Tickets)  
- Ticket Type and Priority Distribution  
- Top Issue Categories  
- Channel Performance Analysis  
- Heatmap (Channel vs Priority)  
- Interactive filters (slicers)  
- Geographic visualization (simulated data)
