## 🎯 Project Overview

This project focuses on analyzing IT support ticket data to evaluate team performance and identify opportunities for operational improvement. Using data analytics and visualization techniques, interactive dashboards were developed to monitor key performance indicators and enhance decision-making.

The project helps in analyzing:

- Ticket Resolution Time  
- Workload Distribution  
- Priority-Based Ticket Handling  
- Team Productivity  
- Customer Satisfaction Metrics  

The primary goal of this project is to improve operational efficiency, reduce ticket resolution time, and optimize IT support team performance using data-driven insights.
