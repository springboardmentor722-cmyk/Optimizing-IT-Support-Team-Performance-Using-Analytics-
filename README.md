This project focuses on analyzing customer support ticket data to understand patterns in ticket handling, customer behavior, and service performance. Using data visualization techniques, the project identifies key trends in ticket types, priorities, channels, and customer satisfaction.
The project also includes an interactive dashboard built using Streamlit for better data exploration.
Analyze customer support ticket dataset
Identify trends in ticket types, priority, and status
Understand customer satisfaction patterns
Visualize data using charts and dashboards
Provide insights to improve support service efficiency
The dataset contains information related to customer support tickets, including:
Ticket ID
Customer Details (Name, Age, Gender)
Product Purchased
Ticket Type
Ticket Priority
Ticket Status
Ticket Channel
Resolution (text-based)
Customer Satisfaction Rating
Technologies Used
Python
Pandas – Data processing
NumPy – Numerical operations
Matplotlib – Data visualization
Streamlit – Interactive dashboard
Jupyter Notebook – Development environment
Analysis & Visualizations
The following analyses were performed:
Ticket Type Distribution
Ticket Priority Distribution
Ticket Status Analysis
Ticket Channel Distribution Customer Satisfaction Analysis
Priority vs Satisfaction Comparison
visualizations include bar charts and comparative analysis to understand trends effectively.
Key Insights
Majority of tickets belong to specific ticket types, indicating recurring issues
Medium priority tickets are most frequent, showing balanced workload
Most tickets are raised through digital channels like Email/Online
Customer satisfaction varies based on ticket handling and priority
Streamlit Dashboard
An interactive dashboard was developed using Streamlit to:
Visualize ticket distribution
Analyze customer satisfaction
Explore trends dynamically
Conclusion
The project successfully identifies important patterns in customer support operations. The insights help in understanding workload distribution, improving service quality, and enhancing customer satisfaction. Data visualization plays a key role in simplifying complex data into actionable insights.
