# Milestone 3 – Cluster and Performance Analysis

## Cluster Analysis

Tickets were grouped into clusters based on resolution performance.

Clusters:

- Fast Resolution Cluster
- Medium Resolution Cluster
- Slow Resolution Cluster
- Pending Cluster

## Similarity Score

Similarity Score = 1 / Resolution Time

This metric helps measure issue similarity based on resolution efficiency.

## Performance Analysis

Analysis performed:

- Resolution time comparison
- Cluster size comparison
- Similarity score distribution
- Priority vs cluster heatmap

## Key Insights

- Fast resolution cluster shows highest efficiency
- Slow resolution cluster indicates performance gaps
- Pending cluster highlights ticket backlog
