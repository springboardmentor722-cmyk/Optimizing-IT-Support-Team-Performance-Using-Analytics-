#  Milestone 3: Cluster & Performance Analysis

##  Overview

This milestone focuses on analyzing issue clusters, measuring similarity between tickets, and evaluating support team performance using data-driven insights.

---

##  Objectives

* Analyze clusters of similar issues
* Measure similarity scores
* Evaluate performance across clusters
* Identify patterns and performance gaps

---

##  Tasks Performed

### 🔹 Cluster Analysis

* Grouped tickets using **Cluster_ID** and **Cluster_Name**
* Identified frequently occurring issue clusters
* Analyzed cluster size and distribution

### 🔹 Similarity Analysis

* Measured issue similarity using **Similarity Score**
* Compared similarity across clusters
* Identified highly similar recurring issues

### 🔹 Performance Analysis

* Analyzed resolution time across:

  * Priority levels
  * Issue categories
* Identified performance gaps in certain clusters
* Evaluated efficiency of issue resolution

---

##  Visual Outputs

###  Key Visualizations

* Cluster distribution charts
* Similarity score analysis
* Resolution time comparison (boxplots)
* Performance scatter plots
